# ⭐ Star Catcher Game

A dark-mode arcade game where you catch green stars and avoid red stars.

---

## 🎮 How to Play

- Move the blue circle using **mouse** or **Arrow Keys (← →)**
- Catch **yellow stars** → +1 point
- Catch **red stars** → -1 life
- Don't let stars hit the bottom!
- Game ends when you lose all 3 lives

---

## 🎯 Objective

Score as many points as possible before losing all lives.

---

## 🕹️ Controls

| Control | Action |
|---------|--------|
| Mouse Move | Move player horizontally |
| ← Arrow | Move left |
| → Arrow | Move right |

---

## 📊 Game Stats

- **Score** – Increases with green/yellow stars
- **Lives** – Starts at 3, decreases when catching red stars

---

## 🛠️ Built With

- HTML5
- CSS3
- Vanilla JavaScript

---

## 🚀 How to Run

1. Save as `index.html`
2. Open in any web browser
3. Click "Start Game" to begin
